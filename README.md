# Financial Calculator

A professional-grade financial calculator application designed to perform complex financial computations with precision and ease. This project features a modern, responsive user interface and a comprehensive suite of financial modules.

> [!NOTE]
> This project was built with the help of **Google Antigravity**.

## Features

- **Time Value of Money (TVM)**: Calculate PV, FV, PMT, Rate, and NPER.
- **Bonds**: Bond valuation, yield to maturity, and duration analysis.
- **Equity**: Stock valuation using various models (e.g., DCF, DDM).
- **Portfolio Optimization**: Modern Portfolio Theory (MPT) calculations and efficient frontier visualization.
- **Loans**: Amortization schedules and loan comparison analysis.
- **Derivatives**: Options pricing using Black-Scholes and other models.
- **Risk Management**: VAR (Value at Risk) and other risk metrics.
- **Macroeconomics**: Indicators and economic trend analysis.
- **Cash Flow**: Analysis of cash flow streams.

## Technology Stack

- **Framework**: [Next.js](https://nextjs.org) (v16)
- **UI Library**: [React](https://react.dev) (v19)
- **Styling**: [Tailwind CSS](https://tailwindcss.com) (v4) & [Shadcn UI](https://ui.shadcn.com)
- **Calculations**: [Math.js](https://mathjs.org)
- **Charts**: [Recharts](https://recharts.org)

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    ```

2.  **Run the development server:**

    ```bash
    npm run dev
    # or
    yarn dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# 金融计算器 (Financial Calculator)

一个专业级的金融计算器应用，旨在精准且便捷地执行复杂的金融计算。本项目拥有现代化、响应式的用户界面，并包含一套全面的金融计算模块。

> [!NOTE]
> 本项目是在 **Google Antigravity** 的帮助下完成的。

## 功能特性

- **货币时间价值 (TVM)**: 计算现值(PV)、终值(FV)、年金(PMT)、利率(Rate)和期数(NPER)。
- **债券 (Bonds)**: 债券估值、到期收益率及久期分析。
- **权益 (Equity)**: 使用多种模型（如 DCF, DDM）进行股票估值。
- **投资组合优化 (Portfolio Optimization)**: 现代投资组合理论 (MPT) 计算及有效前沿可视化。
- **贷款 (Loans)**: 分期偿还计划及贷款比较分析。
- **衍生品 (Derivatives)**: 使用 Black-Scholes 等模型进行期权定价。
- **风险管理 (Risk Management)**: VAR (风险价值) 及其他风险指标。
- **宏观经济 (Macroeconomics)**: 指标与经济趋势分析。
- **现金流 (Cash Flow)**: 现金流分析。

## 技术栈

- **框架**: [Next.js](https://nextjs.org) (v16)
- **UI 库**: [React](https://react.dev) (v19)
- **样式**: [Tailwind CSS](https://tailwindcss.com) (v4) & [Shadcn UI](https://ui.shadcn.com)
- **计算核心**: [Math.js](https://mathjs.org)
- **图表**: [Recharts](https://recharts.org)
